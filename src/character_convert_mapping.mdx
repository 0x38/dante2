# Continuous Blocks

This config will enable the continuous creation of blocks when you create a new line. 
Usually you will not change this config.


## Defaults are : 
```
  character_convert_mapping: {
    '> ': "blockquote",
    '*.': "unordered-list-item",
    '* ': "unordered-list-item",
    '- ': "unordered-list-item",
    '1.': "ordered-list-item",
    '# ': 'header-one',
    '##': 'header-two',
    '==': "unstyled",
    '` ': "code-block"
  }
```
