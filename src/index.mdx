# Editor

## Installation

`npm install --save Dante2@next`

## Basic usage

This is an empty editor. Will be initialized with default tooltips

<Playground>
  <Dante />
</Playground>

Is the same as

<Playground>
  <Dante content={null} />
</Playground>

## Body placeholder

<Playground>
  <Dante body_placeholder={"Do what you will" } 
  />
</Playground>

## Read Only

Read only will disable all interaction

<Playground>
  <Dante read_only={true} />
</Playground>
